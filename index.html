Our Mission

Enterprise procurement remains one of the least automated, most fragmented areas of business operations. Organizations spend 70–85% of their procurement cycles on low-value data entry, status chasing, and manual exception handling. OpenS2P exists to change that — a vendor-neutral, API-first, AI-augmented procurement platform, built in the open, for everyone.

Architecture Principles

Cloud Native

Kubernetes-first, containerized, no bare-metal lock-in.

AI Native

Agents, confidence scoring, and human-in-the-loop gates are first-class, not bolted on.

API First

Every capability is an API before it's a UI.

Event Driven

Domain boundaries crossed only via typed, versioned events.

Enterprise Secure

Zero-trust posture, policy-as-code, PII-safe AI pipelines.

Vendor Neutral

Swap cloud, LLM, or ERP providers — no strategic lock-in.

Open Source Friendly

Apache 2.0, public ADRs, and a low barrier to contribute.
