What is OpenS2P?

OpenS2P is a free, open-source Source-to-Pay platform — the software organizations use to run procurement, from finding and managing suppliers through paying invoices. It's built API-first, event-driven, and AI-native, so any organization can run modern procurement without being locked into a proprietary vendor.
